# 题库理论依据与维度说明

本项目题库为**娱乐性认知推理测评**样例，不等同于标准化 IQ 量表。题目设计参考经典认知测评中的常见维度与题型结构（如数理推理、逻辑推理、言语理解、空间想象），目的是让用户在短时间内体验不同类型的推理任务。

以下为题库的维度划分与理论依据概要：

## 1. 数理推理（Numerical Reasoning）
**核心能力**：数量关系、模式识别、算术推导、函数与比率思维。
**理论基础**：
- 基于“流体智力（Gf）”中的量化推理子能力。
- 常见于 Raven 推理测验、Wechsler 的数理推理题型设计思路。

**对应题目**：1、5、9、13、17

**题型说明**：
- 数列模式识别（等比/差分/非线性变化）
- 文字条件下的速度/效率/比例推导
- 代数恒等式与基础运算

## 2. 逻辑推理（Logical Reasoning）
**核心能力**：命题逻辑、条件推理、三段论、真假话问题。
**理论基础**：
- 演绎推理与形式逻辑是经典认知测评的核心维度之一。
- 与批判性思维、问题解决能力高度相关。

**对应题目**：2、6、10、14、18

**题型说明**：
- 三段论推理（集合包含/蕴含关系）
- 真假话推理（唯一真/唯一假）
- 条件命题的真假判定

## 3. 言语理解（Verbal Reasoning）
**核心能力**：语义关系、类比、同反义辨析、分类能力。
**理论基础**：
- 与“晶体智力（Gc）”中语言与语义理解能力相关。
- 常见于言语类比、词汇理解与分类题型。

**对应题目**：3、7、11、15、19

**题型说明**：
- 词语类比（关系映射）
- 同义/近义辨析
- 异类项识别

## 4. 空间想象（Spatial Reasoning）
**核心能力**：空间变换、旋转、镜像、图形构成与结构关系。
**理论基础**：
- 空间能力是经典智力结构模型中的重要维度。
- 常见于空间旋转、折叠与视觉化推理题型。

**对应题目**：4、8、12、16、20

**题型说明**：
- 正方体展开/折叠与对面关系判断
- 图形旋转与镜像的等价关系
- 位置变换与空间结构保持

---

## 使用提示
- 题库为示例题，主要用于演示与娱乐体验，不作为专业诊断。
- 若希望提高科学性与稳定性，可扩充题库并进行样本验证与统计校准。
