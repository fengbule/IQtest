# 题库理论依据与维度说明

本项目题库为**娱乐性认知推理测评**样例，不等同于标准化 IQ 量表，也不能替代医院、学校、心理机构使用的正式测验。题目设计主要参考认知能力研究里较常见的能力维度划分方式，例如：

- **CHC（Cattell-Horn-Carroll）认知能力模型**中常见的广义能力分类
- **流体智力（Gf）**：更偏向新问题解决、规律发现、抽象推理
- **晶体智力（Gc）**：更偏向语言理解、词汇、语义与已习得知识运用
- **视觉空间能力（Gv）**：更偏向空间旋转、图形结构、视觉化推理

本项目的目的，是让用户在较短时间内感受不同类型的认知任务，并获得一个**站内内部换算的粗略参考结果**，而不是形成正式诊断结论。

以下为题库的维度划分与理论依据概要：

## 1. 数理推理（Numerical Reasoning）
**核心能力**：数量关系、模式识别、算术推导、函数与比率思维。
**理论基础**：
- 与 **流体智力（Gf）** 中的定量推理、抽象规律发现有关。
- 在经典智力研究和能力测验中，数列、比例、速度效率、数量模式等任务常被视作观察推理效率的一种方式。
- 这类题更关注“是否能从陌生信息中发现规律”，而不是单纯背诵知识点。

**对应题目**：1、5、9、13、17

**题型说明**：
- 数列模式识别（等比/差分/非线性变化）
- 文字条件下的速度/效率/比例推导
- 代数恒等式与基础运算

## 2. 逻辑推理（Logical Reasoning）
**核心能力**：命题逻辑、条件推理、三段论、真假话问题。
**理论基础**：
- 与 **流体智力（Gf）** 中的演绎推理、关系推理、规则应用有关。
- 常见于条件命题、集合包含、真假判断、结论是否成立等题型。
- 这类题主要观察一个人在信息有限时，能否保持推理步骤清晰、不被表面文字干扰。

**对应题目**：2、6、10、14、18

**题型说明**：
- 三段论推理（集合包含/蕴含关系）
- 真假话推理（唯一真/唯一假）
- 条件命题的真假判定

## 3. 言语理解（Verbal Reasoning）
**核心能力**：语义关系、类比、同反义辨析、分类能力。
**理论基础**：
- 与 **晶体智力（Gc）** 中的词汇、语言理解、概念关系、语义提取有关。
- 常见于类比、近义辨析、概念分类、词义关系等任务。
- 这类题往往既受语言理解能力影响，也会受教育背景、阅读经验和词汇积累影响。

**对应题目**：3、7、11、15、19

**题型说明**：
- 词语类比（关系映射）
- 同义/近义辨析
- 异类项识别

## 4. 空间想象（Spatial Reasoning）
**核心能力**：空间变换、旋转、镜像、图形构成与结构关系。
**理论基础**：
- 与 **视觉空间能力（Gv）** 中的心理旋转、图形表征、结构保持有关。
- 常见于立体折叠、图形旋转、镜像判断、空间位置变化等任务。
- 这类题通常更依赖视觉化操作能力，而不是语言知识。

**对应题目**：4、8、12、16、20

**题型说明**：
- 正方体展开/折叠与对面关系判断
- 图形旋转与镜像的等价关系
- 位置变换与空间结构保持

## 结果是如何理解的
本项目结果页会展示：

- **参考 IQ**：基于本站题库与内部换算模型得到的粗略参考值
- **百分位**：表示本次结果在站内内部换算中大致处于什么位置
- **维度表现**：展示数理、逻辑、言语、空间四类题中的相对强弱项

这些指标的意义更接近：
> “你在这套题上，本次大致表现如何、相对强弱项在哪里、后续该先练什么”

而不是：
> “你已经被正式诊断为某个固定智力水平”

## 为什么不能把结果当成正式 IQ 诊断
在线非标准化测验通常存在以下限制：

1. **题量有限**：题目数量较少时，单次发挥会明显影响结果。
2. **样本未严格校准**：如果没有大规模代表性样本做常模，分数只能算内部参考。
3. **设备与环境影响较大**：网络、屏幕、输入设备、注意力状态都会影响表现。
4. **存在练习效应**：做过类似题型后，再测常会更熟悉，不完全代表能力真实增长。
5. **受教育与文化背景影响**：尤其是言语理解题，更容易受到阅读经验和知识背景影响。

因此，本项目更适合用来做：

- 初步体验
- 自我观察
- 看看自己的强项和短板
- 做前后练习对比

而不适合用于：

- 医学诊断
- 招聘筛选
- 升学评估
- 对个人能力做绝对定性判断

## 参考区间的谨慎表述方式
为避免误导，本项目建议使用以下更谨慎的结果表述：

- “当前表现大致处于本测验内部的常见区间”
- “从这次作答看，数理推理相对更强”
- “本次结果更适合作为摸底参考，不建议直接视为稳定能力结论”
- “若未完整作答或提交过快，结果可能会偏低”

## 若后续想提高科学性，可以继续做什么
如果后续希望把项目做得更接近严谨测评，可以逐步增加：

1. 更大的题库与难度梯度
2. 更多样本数据与常模校准
3. 对重复测验的练习效应控制
4. 对不同设备、答题时间、完成度的修正
5. 更清晰的区间解释和结果置信提示

## 可引用的公开理论线索
- Cattell-Horn-Carroll (CHC) cognitive abilities framework
- Fluid intelligence (Gf) / Crystallized intelligence (Gc)
- Visual-spatial ability / spatial reasoning
- 在线数字认知测评研究中常见的限制：样本代表性、设备差异、练习效应、远程环境控制不足

## 使用提示
- 题库为示例题，主要用于演示与娱乐体验，不作为专业诊断。
- 如果想让结果更接近真实水平，建议完整作答、避免过快提交，并结合错题复盘再次测试。
- 若希望提高科学性与稳定性，可扩充题库并进行样本验证与统计校准。
